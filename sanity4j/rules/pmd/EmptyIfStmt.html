EmptyIfStmt

Empty If Statement finds instances where a condition is checked but nothing is done about it.

This rule is defined by the following XPath expression:


//IfStatement/Statement
 [EmptyStatement or Block[count(*) = 0]]

Example:

                
    
public class Foo {
 void bar(int x) {
  if (x == 0) {
   // empty!
  }
 }
}

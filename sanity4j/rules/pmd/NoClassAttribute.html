NoClassAttribute

Since: PMD 3.6

Do not use an attribute called 'class'. Use "styleclass" for CSS styles.

This rule is defined by the following XPath expression:

					 //Attribute[ string:upper-case(@Name)="CLASS" ]

Example:

                
			
<HTML> <BODY>
<P class="MajorHeading">Some text</P>
</BODY> </HTML>

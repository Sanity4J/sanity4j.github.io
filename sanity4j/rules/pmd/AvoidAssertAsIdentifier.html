AvoidAssertAsIdentifier

Since: PMD 3.4

Finds all places where 'assert' is used as an identifier.

This rule is defined by the following XPath expression:

                  
//VariableDeclaratorId[@Image='assert']

Example:

                
  
    public class A {
        public  class foo {
            String assert = "foo";
        }
    }

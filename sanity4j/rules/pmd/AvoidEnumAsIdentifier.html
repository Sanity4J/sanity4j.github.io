AvoidEnumAsIdentifier

Since: PMD 3.4

Finds all places where 'enum' is used as an identifier.

This rule is defined by the following XPath expression:

                  
//VariableDeclaratorId[@Image='enum']

Example:

                
  
    public class A {
        public  class foo {
            String enum = "foo";
        }
    }

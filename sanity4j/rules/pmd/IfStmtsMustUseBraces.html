IfStmtsMustUseBraces

Avoid using if statements without using curly braces.

This rule is defined by the following XPath expression:

                   
//IfStatement[count(*) &lt; 3][not(Statement/Block)]

Example:

                
 
 public class Foo {
   public void bar() {
     int x = 0;
     if (foo) x++;
   }
 }

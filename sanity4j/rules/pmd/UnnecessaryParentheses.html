UnnecessaryParentheses

Sometimes expressions are wrapped in unnecessary parentheses, making them look like function calls.

                  
          //Expression
           /PrimaryExpression
            /PrimaryPrefix
             /Expression[count(*)=1]
              /PrimaryExpression
              /PrimaryPrefix

Example(s):

  
public class Foo {
   boolean bar() {
      return (true);
      }
}
